сервер работает на http://92.242.58.199:5010/web
изменить данные можно в файле config.json
	"ip_listen": "0.0.0.0",
	"port_listen": 5010

команды запуска сервера (из дирректории .../node_express_f):
(запускает сервер, пока открыта консоль) 
	node app.js 
если не выключился при закрытии консоли то: 
	ps aux | grep node
	kill -9 PROCESS_ID (где PROCESS_ID - 2й столбец слева от работающего процеса node) 

(демонизирует сервер, при закрытии консоли он продолжит работать) 
	pm2 start app.js
остановить демонизацию
	pm2 stop app.js
посмотреть список демонизированных приложений
	pm2 list или pm2 ls

Функции сайта:
	Применить 
		- переносит все файлы данной серии кроме отмеченных («не добавлять») из 1й во 2ю дирректорию в папку пользователя который выбирается в строке ввода в серии. 
		- После переноса удаляет папку этой серии со всем содержимым из 1й дирректории 
	Удалить серию 
		- удаляет из 1й дирректории папку с данными фотографиями 
	Добавить человека
		- добавляет человека в people_json.json файл в корне проекта 
		- создает папку с названием хэша этого человека в дирректориях 2й и 3й

	Быстрый выбор:
		начать выбирать 
			- скраывает обычные блоки управления сериями и показывает checkbox выбора серий
			- при каждом выборе чекбокса изменяет показатель "Отмечено"
			- изменяет назначение кнопки на "отменить выбор"
				отменить выбор
					- возвращает интерфейс к обычному виду
		сохранить
			- применяет к каждой выбранной серии фотографий действия функции "применить" с именем человека выборанным в input быстрого меню
			- возвращает интерфейс к обычному выбору

Обновления Ajax:
	все текстовые поля ввода ФИО человека (кроме поля в "Добавить человека")
		- обновление выпадающего списка происходит на вводе 2го символа слова
		- далее пришедший имеющийся список вариантов сам сортируется на клиенте при дальнейшем вводе букв
	поле в "Добавить человека"
		- проверка осуществляется на каждом 4 или более введенном символе 
		- если такое имя уже существует в people_json.json то выводит ('Такое имя уже есть')
		- если такого имени не найдено в people_json.json то выводит ('Такое имя можно внести')
	действия с сериами (применить/удалить серию/быстрый выбор)
		- отправляется ajax запрос на сервер 
		- после ответа сервера происходят изменения на клиенте (в браузере): 
			- скрывается обработанная серия 
			- обновляется число "Непознанные фотографии"
			- обновляется чисто "Отмечено" если был быстрый выбор
